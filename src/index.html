Magic Brain Face Finder

A full-stack application which allows a user to create an account, then add an image URL to the page to determine how many faces are in the user added picture. The page then also stores the amount of faces a user has detected total. Created using a React frontend that pulls from the Clarifai Face Detection API and sends information back and forth to a backend which consists of an Express.js server and a PostgreSQL database.
